== Description ==

This plugin has a few tools, primarily to advertise our service, provide Spotlight Branding throughout the client experience, and provide some useful tools that we integrate into most of our websites.

== Installation ==

 1. Install plugin by clicking Pluings > Add New
 2. upload the plugin by clicking Upload plugin button
 3. Select the pluign zip file to be install
 4. Activate the plugin by clicking activate pluign 
 
== Access Plugin Settings  ==

  1. Access the pluign settings page by going to Tools > SS Toolkit 2.0

== Features ==

- Custom Admin Login Page
- Shortcodes can be used in Website
- A Dashboard Widgets consists of plugin developer site details 
- Manage permission to access the plugin Page
- Option to add Google analytics to the site
- Option to manage RSS Feed details display in login Page
- Option to manage custom login page background image
- Option to prevent plugin deactivation option
